# Тест-кейсы
- Создать объявление

1. Успешное сохранения объявления
   + Ожидаемый результат ответа 200
   + Все поля совпадают с отправленными
2. Отрицательные кейсы
    + При отсутствии тела запроса вывести 400
- Получить объявления по его идентификатору
1. Успешные кейсы
    + Вывод соответствующего ответа
    + Вывод кода 200
2. Отрицательные кейсы 
    + Невалидный запрос, вывести код 400
    + Не нашел такого объявления, вывести код 404
    + Ошибка на стороне сервера, вывести 500
- Получить все объявления по идентификатору продавца
1. Успешные кейсы
    + Вывод соответствущих объявлений
    + Вывод кода 200
2. Отрицательные кейсы
    + Невалидный запрос, вывести код 400
    + Нет такого продавца, вывести код 400
    + Ошибка на стороне сервера, вывести 500
- Получить статистику по айтем id
1. Успешные кейсы
    + Вывод соответствущих объявлений
    + Вывод кода 200
2. Отрицательные кейсы
    + Невалидный запрос, вывести код 400
    + Нет такой статистики, вывести код 400
    + Ошибка на стороне сервера, вывести 500
